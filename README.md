# 🍽️ FoodPoint – Recipe Website

A modern, responsive **Food Recipe Web Application** built using HTML, CSS, and JavaScript.
This project allows users to explore recipes, view detailed cooking instructions, and even add their own recipes dynamically.

---

## 🚀 Features

* 💜 Modern UI with purple theme
* 🍝 Animated hero section (floating food + glow effects)
* 📱 Fully responsive (mobile + desktop)
* 🍽️ View detailed recipes (ingredients + steps)
* ➕ Add new recipes dynamically
* 💾 Data stored using LocalStorage (no backend required)
* 🎯 Smooth animations and interactive UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (DOM, LocalStorage)

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/foodpoint.git
```

2. Open the project folder

3. Run `index.html` in your browser

---

## ✨ Usage

* Click **“View More”** to see full recipe details
* Click **“+ Add Recipe”** to add your own recipe
* Data is automatically saved in your browser
